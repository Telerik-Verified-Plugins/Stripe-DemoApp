Stripe Plugin Demo

The Stripe plugin offers easy access to the Stripe REST-based API.

Because the API has similar methods for interaction with the various objects, this demo only shows how to perform CRUD operations on the Customers object.

Important
Make sure you specify the API_KEY parameter in the plugin settings.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android - App package.

Customers

Here you can try a few CRUD operations on the Customers object. Make sure you're connected to a test account so no actual Customers are affected.
